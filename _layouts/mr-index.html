Memory Retention

Sets of repeat sentence exercises to help develop memory retention

Instructions: This is a memory retention exercise, also referred to as repeat sentence exercise.  The exercise uses pre-recorded audio materials, where each recording contains a set of sentences in English.  To do the exercise, start by playing a recording.  After each sentence, try to retain what was said and then repeat it exactly as is in English.  Do not add anything or omit anything.  The recorded materials have built-in pauses after each sentence.  These pauses allow for adequate time to repeat the sentence.

For additional practice, record yourself as you are doing the exercises. Then, after you have finished a set of sentences, listen to your recording and compare it to the text that accompanies each recording. To extend the exercise even further, render an interpretation of the sentences orally into your target language (which gives your additional practice for consecutive interpretation), and then perform a written translation of the provided text as well.
