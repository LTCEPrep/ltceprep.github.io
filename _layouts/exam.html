LTC Exam

Information about the state of Washington's DSHS LTC Exam for Medical Interpreters

Part of the LTC Exam for Medical Interpreters is an oral section.  The structure of this section of the exam differs depending on the target language of the exam—that is, the language for which the test taker wants to be certified or authorized to work as a medical interpreter.  Some will receive one version of the exam, which will have its own set of oral exercises.  Others will receive a different version, which will similarly have its own set of oral exercises.

How does this work?  The State of Washington's DSHS divides target languages into two primary categories—certified and screened. Eight languages are classified as certified.  These are Cambodian, Chinese-Cantonese, Chinese-Mandarin, Korean, Laotian, Russian, Spanish, and Vietnamese.  All other languages are classified as screened languages. Which version of the test administered depends on into which category the target language falls.

How do the tests differ?  Both versions of the exam have a written and an oral section. However, the exam for certified languages will be in both a target language and in English, while the exam for screened languages will only be in English.  How the written and oral sections vary according to language group can be seen in Table 1 below. And how the oral exam is scored can be seen in Table 2 below.

Written Section

The written test for medical interpreters has five sections, all in multiple choice formats.

  • Section one — professional code of ethics: In this section, statements regarding professional code of conduct are written in the English language, and the candidate will need to determine whether each statement is true or false.

  • Section two — medical terminologies: This section covers such areas as parts of the body, symptoms, disease/illness/injury/physical disorder, treatment, prescription, medical personnel, and miscellaneous health related expressions. All stems in section two are in the English language while all options are in a non-English language. For each question, the candidate will need to identify the target language equivalent of the English word/term in the stem.

  • Section three — clinical/medical procedures: Both stems and options in this section are in the English language. The candidate will need to choose from those given options the one that best describes each procedure.

  • Section four — brief writing test in the English language: The candidate will need to choose an option to best complete each unfinished sentence.

  • Section five — brief writing test in a non-English language: The format of this section is exactly the same as that of Section four, except that all items are written in a language other than English.

The written screening test for medical interpreters is composed of four sections, all in multiple choice formats:

  • Section one: professional code of conduct. In this section, statements regarding professional code of conduct are written in the English language, and the candidate will need to determine whether each statement is true or false.

  • Section two: medical terminologies; This section covers such areas as parts of the body, symptoms, disease/illness/injury/physical disorder, treatment, prescription, medical personnel, and miscellaneous health related expressions. All stems in section two are in the English language AND all options are in English. For each question, the candidates will need to make the best choice from four options provided.

  • Section three: clinical/medical procedures; Both stems and options in this section are in the English language. The candidate will need to choose from those given options the one that best describes each procedure.

  • Section four: indirect writing test in the English language. The candidate will need to choose an option to best complete each unfinished sentence.

Note: All items in the written screening test are in English.
Oral Section

The oral test for medical interpreters is composed of two parts — sight translation and consecutive interpretation.

  • Part one (A+B)— sight translation: The candidate will be allowed six minutes to (A) orally render an English text into a target language, and another six minutes (B) to orally render a non-English language text into English. The candidate may silently review each text before interpreting, but the review time will be part of the six minutes permitted for each text.

  • Part two — consecutive interpretation: Some pre-recorded audio materials will be played in this exercise. The candidate will assume the role of an interpreter. The recorded materials have built-in pauses. The pauses allow adequate time for the candidate to render the interpretation. Note: Interpretation will go from English into the candidate's target language AND from the candidate's target language into English.

The entire oral exam will be audio-recorded for scoring accuracy and documentation. The recorded test will be scored by independent raters retained by LTC, who are highly skilled professionals and/or certified interpreters.

The oral screening test for medical interpreters has three parts.

  • Part one — sight translation. The candidate will orally render ten unrelated sentences from English into a target language. The candidate may silently review each text before interpreting, but it will part of the time allotted to render the sentences.

  • Part two — memory retention

  • Part three — consecutive interpretation. The candidate will have to provide a consecutive interpretation from a target language into English. Some pre-recorded audio materials will be played in this exercise. The recorded materials have built-in pauses. The pauses allow adequate time for the candidate to render the interpretation.

The oral screening test will be audio recorded for the purpose of scoring and record keeping. Objective scoring will be the only method employed in evaluating the oral screening test.
