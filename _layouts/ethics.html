Code of Ethics

• Study the Code

The state of Washington's DSHS has a Code of Professional Conduct for Interpreter, Translator, and Licensed Agency Personnel. This code comprises the obligations that medical interpreters must honor when carrying out their duties. These 12 core pinciples of the profession provide a guide for behavior which is expected to be followed.

NOTE: The LTC exam for medical interpreters has a number of questions concerning the aforementioned code ethics. Accordingly, it is incumbent on the test taker to study the code prior to the exam if to do well on the written portion.

A PDF containing the code of ethics is available on the state of Washington's DSHS website (scroll to bottom of page).

Ethics Quiz

• Test knowledge

The ethics quiz created by the staff at LTCE Prep comprises 12 'True or False' questions covering the 12 core principles of the state of Washington's DSHS Code of Professional Conduct for Interpreter, Translator, and Licensed Agency Personnel. Our quiz is attempts to mimic the state exam—more or less—thereby affording individuals a chance to check their comprehension of the ethical principles prior to the actual LTC exam.

Results of the exam are provided immediately to the test taker. Additionally, the results will be sent as anonymous data to staff of LTCE Prep. That is, no personal identifying information will be recorded.

WA DSHS Video

• Watch video

The state of Washington's DSHS provides an orientation and ethics training video for medical interpreters intended for individuals who have passed the written and oral sections of the state exam. Nevertheless, anyone may watch the video, regardless of whether he or she as passed yet to pass the exam may still watch the video.

The video goes into some detail regarding the code of ethics, perhaps more than what may be required to prepare for the state exam. Evenstill, a watching helps to enhance an understanding of the code by putting them into some context.
